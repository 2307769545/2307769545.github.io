# 欢迎来到我的博客

大家好，这是我用 Jekyll 和 GitHub Pages 搭建的个人博客。

## 关于这次作业

通过这次开源软件开发实践，我学习了如何使用 GitHub Pages 搭建个人博客网站。主要步骤包括：

1. 创建 GitHub 仓库
2. 配置 Jekyll 博客环境
3. 编写 Markdown 格式的博客文章
4. 部署到 GitHub Pages

## 心得体会

这次实践让我对静态网站生成器和 GitHub Pages 有了初步的了解，也掌握了 Markdown 的基本用法。后续我会继续更新博客，记录学习过程中的收获。
